# Калькулятор

## Документація для користувача

Щоб отримати результат, потрібно спочатку ввести вхідні данні (числа) у поля "Value 1" та "Value 2", після  
цього потрібно натиснути на одну із кнопок нижче щоб обрати тип обчислень.
